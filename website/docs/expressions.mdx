:::info
Para sobreescribir la precedencia de los operadores se puede hacer uso de los parentesis
:::

## Ejemplos

```
fA = iA * (iB + iC * iD - fA) > iB + (iC * iD - fA);

/*
  Dentro de las expresiones se pueden hacer llamadas a
  funciones y accesos a elementos de vectores o matrices
*/

println(dos((i + k) * i / 4 - 7, f * (3 - 2 + 1)) + 3);

c[i][j] = c[i][j] + (a[i][k] * b[k][j]);
```

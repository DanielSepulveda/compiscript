## Declaracion de funciones

Las funciones declaradas siguen la siguiente estructura:

```
<tipo de retorno> function <nombre> (<lista de parametros>)
<declaracion de variables locales>
{
  ...
}
```

Ejemplo:

```
int function fib(n: int)
{
  if (n < 2) {
    return (n);
  } else {
    return (fib(n - 1) + fib(n - 2));
  }
}

void function test(m: string, t: int)
vars
  i: int;
{
  for (i = 0) until (t - 1) {
    println(m);
    i = i + 1;
  }
}
```

## Parametros

Se pueden especificar cero o mas parametros. Cada parametro tiene que especificar su tipo.

Ejemplo:

```
int function sum(a: int, b: int) {
  return (a + b);
}

float function returnPI() {
  return (3.1416);
}
```

## Retorno

Las funciones que no sean de tipo <code>void</code> deben de regresar un valor del mismo tipo de la funcion declarada. La estructura de los estatutos de retorno es la siguiente:

```
return (<expresion>)
```

Ejemplo:

```
int function fib(n: int)
{
  if (n < 2) {
    return (n);
  } else {
    return (fib(n - 1) + fib(n - 2));
  }
}
```
